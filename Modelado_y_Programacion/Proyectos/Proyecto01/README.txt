Forma de utilizar el programa:
    
    Este proyecto utiliza una interfaz gráfica de usuario con la biblioteca 'Tkinter'.
    Por ello, para ejecutar el proyecto es necesario tenerlo instalado en su computadora.

    Documentación de Tkinter: https://docs.python.org/3/library/tkinter.html
    Línea de comando para instalar Tkinter para Python3: sudo apt-get install python3-tk
    
    En la terminal de Linux ejecutar el siguiente comando:
    
        >>> python3 proyect_one.py
    
    Se abre la interfaz que da inicio al progama.
